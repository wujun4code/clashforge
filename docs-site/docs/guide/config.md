# 添加代理订阅

安装完成后，ClashForge 还不知道你要使用哪个代理服务。  
你需要把自己的订阅链接或配置文件添加进去。

## 你需要准备什么

常见有两种：

| 你手里的东西 | 怎么使用 |
| --- | --- |
| 订阅链接 | 复制链接，粘贴到 ClashForge 的订阅设置里 |
| Clash 配置文件 | 上传 `.yaml` / `.yml` 文件，或把内容粘贴进去 |

如果你不知道自己有没有订阅链接，通常需要从代理服务商的后台复制。

## 推荐新手用订阅链接

订阅链接更适合长期使用，因为服务商更新节点后，你可以在 ClashForge 里重新更新订阅。

建议：

1. 给订阅起一个容易看懂的名字，例如“主订阅”。
2. 不要把订阅链接发到群里、截图里或 Issue 里。
3. 如果有备用订阅，也可以一起添加。

## 第一次配置怎么做

打开：

```text
http://192.168.20.1:7777
```

然后按顺序操作：

1. 进入 Setup 或配置页面。
2. 添加订阅链接，或上传配置文件。
3. 保存。
4. 选择刚添加的配置。
5. 点击启动代理服务。

启动后先看页面上是否有节点、连接状态、运行状态。

## 什么是“配置文件”

你可以把配置文件理解成代理使用说明书。它通常包含：

1. 有哪些代理节点。
2. 哪些网站需要走代理。
3. 哪些网站直接访问。

ClashForge 做的事情，是把这份说明书交给路由器上的代理程序去执行。

## 第一次不要改太多

第一次只建议做这几件事：

1. 添加一个订阅或配置。
2. 启动代理服务。
3. 确认常用网站能访问。

先不要急着改高级网络选项。等你确认基本可用后，再慢慢调整。

## 怎么知道配置成功了

满足这些条件即可：

1. Web 页面能打开。
2. 页面里能看到节点或策略组。
3. 启动代理服务后没有持续报错。
4. 常用需要代理的网站能访问。

如果不满足，先看 [不好用怎么办](/guide/troubleshooting)。
